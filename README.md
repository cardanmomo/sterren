# Sterren
This repository includes all the routines and files necessary to analyse the ZF/GAM response linear and nonlinearly
Key:
File name: @@@**s##

@@@:device name (W7X, LHD, Sa)
**: configuration
For Wendelstein 7-X, this can be:
hi: High Iota
li: Lowe Iota
hm: High Mirror
lm: Low Mirror
sd: Standard Configuration
